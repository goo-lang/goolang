# Details:
Implement the type system and type checker that enforces type safety, including Goo's ownership tracking, nullable types, and error unions.

Type System Components:
```go
type Type interface {
    String() string
    Size() int
    Align() int
}

type OwnershipKind int
const (
    Owned OwnershipKind = iota
    Borrowed
    Shared
)

type QualifiedType struct {
    Base      Type
    Nullable  bool
    Ownership OwnershipKind
}

type ErrorUnionType struct {
    ValueType Type
    ErrorType Type  // Usually a standard error type
}

type ChannelType struct {
    Pattern  ChannelPattern
    ElemType Type
}
```

Type Checker Implementation:
1. Create a TypeChecker struct that traverses the AST
2. Implement type inference for expressions
3. Verify type compatibility in assignments and function calls
4. Check ownership rules (simplified from Rust)
5. Enforce null safety with nullable types
6. Validate error union handling
7. Verify channel pattern usage

Ownership Checking:
- Track variable ownership through scopes
- Detect use-after-move errors
- Implement borrowing rules
- Generate clear error messages for ownership violations

Null Safety:
- Ensure nullable types are checked before use
- Track null state through control flow
- Prevent null pointer dereferences
